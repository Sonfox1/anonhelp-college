AnonHelp College — ANON CORE FINAL
Тема: Веб-ориентированная информационная система анонимного взаимодействия и взаимопомощи обучающихся колледжа.

Главный принцип версии:
ВСЁ ВЗАИМОДЕЙСТВИЕ ОБЕЗЛИЧЕНО.
Даже администратор не видит, кто отправил обращение или комментарий. В админ-панели отображаются только номер обращения, категория, статус, приоритет, текст, дата, жалобы и статистика. ФИО, email, группа и логин автора скрыты.

Запуск:
1) Открыть папку AnonHelp_College в VS Code.
2) В терминале выполнить:
   py -m pip install -r requirements.txt
   py app.py
3) Открыть в браузере:
   http://127.0.0.1:5000

Демо-доступ:
Администратор: admin / admin123
Студент: student / student123

Модули:
- регистрация и авторизация;
- личный кабинет студента;
- анонимная подача обращения;
- публичная лента взаимопомощи;
- анонимные комментарии;
- отметка лучшего ответа;
- лайки/поддержка;
- жалобы модератору;
- уведомления;
- загрузка файлов к обращению;
- обезличенная админ-панель;
- статистика по категориям и статусам;
- протокол полной анонимности;
- светлая и темная тема.

Вишенка проекта:
Протокол полной анонимности. Система показывает, что администрация может обрабатывать обращения и видеть аналитику, но не может определить личность автора обращения или комментария.

Если после обновления появляется ошибка базы данных:
Удалите файл instance/anonhelp.db и запустите проект заново.
